PV String toevoegen

Een PV-string is altijd fysiek verbonden aan een specifiek kanaal op de omvormer. Het is essentieel om de juiste string aan het corresponderende kanaal toe te voegen voor een correcte monitoring en sturing.

Meerdere strings

Heeft de omvormer meerdere strings (bijv. één op het oosten en één op het westen)? Dan moet u elke string apart toevoegen via de + Solar Array knop. Zo kan het systeem de opbrengst per kanaal en per windrichting nauwkeurig berekenen.

Hoe voegt u een PV String toe?​

  1. Klik in de boomstructuur op de knop + Solar Array.

  1. Selecteer aan welk Device deze string fysiek gelinkt is.

  1. Vul de configuratieparameters in.

Technische parameters​

  • Name: Geef de string een herkenbare naam (bijv. "Panelen zuid").
  • Latitude & Longitude: De geografische coördinaten van de locatie.
  • Tilt (°): De hellingshoek van de panelen.
  • Azimuth (°): De richting waarin de panelen liggen ten opzichte van het noorden.
  • Power (peak) (kWp): Het maximale vermogen dat de panelen in deze string kunnen opwekken.
  • Power Max (kW): Het maximale vermogen dat de omvormer toelaat.
  • Terrain shading: Zet dit op True indien er sprake is van schaduwvorming (bijv. door bomen of een schouw).
  • Loss factor (%): Deze factor wordt initieel bepaald door de installatiedatum. Indien de productie consistent lager ligt dan de voorspelling, kan deze factor handmatig verhoogd worden om de voorspelling bij te sturen.
  • Install date: De datum waarop de panelen in gebruik zijn genomen.
  • Channel: Selecteer het fysieke kanaal op de omvormer waarop deze string is aangesloten.

De richting (Azimuth) bepalen​

De Azimuth wordt gemeten van -180° tot 180° ten opzichte van het noorden:

  • 0° = Noord | -90° = Oost | 90° = West | 180° = Zuid

Rekenvoorbeeld: Het kompas geeft 270° aan. Berekening: 270 - 360 = -90. Vul in: -90

Status​

Na het opslaan gebruikt Helios deze data voor de opbrengstvoorspelling. Door de Loss factor en Terrain shading accuraat in te vullen, optimaliseert u de nauwkeurigheid van het energiebeheersysteem.
